# 🦷 Dentist Appointment Booking Platform

## 📌 Project Overview

This is a full-stack web application that allows users to view dentists and book appointments online.

## 🚀 Features

* View list of dentists
* Book appointment
* Prevent duplicate bookings
* Admin panel to view all appointments
* Form validation
* Date restriction (no past dates)

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB
* Styling: CSS

## ⚙️ Setup Instructions

### Frontend

```bash
cd dentist-appointment
npm install
npm start
```

### Backend

```bash
cd backend
npm install
node server.js
```

## 🌐 API Endpoints

* GET /api/appointments
* POST /api/appointments
